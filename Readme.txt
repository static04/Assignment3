# Hollywood Movies App

Welcome to the Hollywood Movies App! This is a simple web application built with Node.js, Express, MongoDB, and Bootstrap. The app allows users to manage a list of Hollywood movies, including adding new movies, updating existing ones, and deleting movies. It comes with a user-friendly interface and supports CRUD operations (Create, Read, Update, Delete).

## Features

- **View Movies:** Browse a list of Hollywood movies with details such as title, genre, release year, and rating.
- **Add Movie:** Easily add a new movie to the collection by filling out a simple form.
- **Update Movie:** Modify the details of an existing movie, such as title, genre, release year, and rating.
- **Delete Movie:** Remove a movie from the collection with a click of a button.

## Getting Started

1. Clone the repository to local machine.
2. Install dependencies using `npm install`.
3. Run the backend server using `npm start`.

## Technologies Used

- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
